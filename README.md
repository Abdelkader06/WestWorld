# WestWorld

## c'est un jeux console qui comporte une course poursuite au milieu du désert.

## Le jeu consiste donc à diriger (grâce aux flèches directionnelles) une diligence sur un chemin
à 3 voies, des obstacles apparaissent aléatoirement sur les voies. Si la diligence heurte un des
obstacles, c’est perdu. 

**Langage :** C#
**Framework :** .net 5.0
**IDE :: ** Visual Studio.
**SVM :: ** Githube.
